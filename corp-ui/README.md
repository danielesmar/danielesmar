# Corporate UI Theme - Oracle APEX

## Instalación

1. Subir carpeta corp-ui al servidor (ORDS static o Static Application Files).
2. Agregar en User Interface > CSS:
   /corp-ui/css/corp-ui.css
3. Agregar en JavaScript:
   /corp-ui/js/theme-switcher.js

## Cambiar colores corporativos
Modificar únicamente:
css/01-variables.css
